选中即用

Select any text on your Mac and a popup appears instantly — copy, search, translate, open links, look up in the dictionary, and more. Smart enough to show only what makes sense: Cut hides in read-only text, Translate hides on URLs and paths.

在 Mac 上选中任意文字，弹窗立刻出现——复制、搜索、翻译、打开链接、系统字典查词，一触即发。足够聪明：只读内容不显示剪切，URL 和路径不显示翻译。

Features

功能

Everything you need.
Nothing you don't.

刚好够用，绝不多余。

Built for real daily use. Every action is context-aware — it shows only when it actually makes sense.

为真实日常使用设计。每个动作都感知上下文——只在真正合适的时候出现。

New in v0.5 v0.5 新增
  • Left-Click Boost — hold ⌥ Option and click or long-press to fire custom actions: send a shortcut, open an app, or run a Shortcut workflow. Three hold-duration tiers.
  • Middle-Click Boost — bind the scroll-wheel click to custom actions, with ⌥ Option variants. Off by default.
  • Run Shortcut — pick from your existing Shortcuts in a dropdown; runs silently in the background, no app window.
  • 左键加强 — 按住 ⌥ Option 单击或长按，触发自定义动作：发送快捷键、打开 App 或运行 Shortcuts。三档时长。
  • 中键加强 — 将滚轮键绑定自定义动作，支持 ⌥ 组合。默认关闭。
  • 运行快捷指令 — 从下拉列表直接选择快捷指令，静默后台运行，不打开 Shortcuts.app。

Changelog

更新日志

Release history.

版本历史

v0.5.1 2026-06-28 Settings UI rebuilt — cleaner layout, clearer hierarchy, collapsible Mouse Enhancement panels. 设置界面重构——更清晰的布局层级，鼠标增强面板默认折叠。
v0.5.0 2026-06-28 Middle-Click Boost — bind scroll-wheel click/long-press to custom actions (⌥ variants supported). Off by default. 中键加强——滚轮键单击/长按触发自定义动作，支持 ⌥ 组合，默认关闭。
v0.4.0 2026-06-27 Left-Click Boost upgraded to 3 hold-duration tiers. New "Run Shortcut" action — pick from dropdown, runs silently in background. 左键加强升级三档时长，新增「运行快捷指令」动作，下拉选择，后台静默运行。
v0.3.0 2026-06-27 Left-Click Boost — ⌥ + click/long-press fires custom actions. Plain clicks untouched. 左键加强——⌥+单击/长按触发自定义动作，普通点击完全不受影响。
v0.2.3 2026-06-27 Removed Paste action — it never belonged in a selection-triggered popup. ⌘V already handles it. 移除「粘贴」动作——粘贴作用于光标而非选区，⌘V 本就搞定。
v0.2.2 2026-06-27 Cut now hides in read-only contexts. Translate and Dictionary hide on URLs and file paths. 只读内容不显示剪切；URL 和路径不显示翻译与查词。
v0.1.0 2026-06-26 First release — copy, search, translate, dictionary, open link, Finder path, exclusions, bilingual UI. 首个版本——复制、搜索、翻译、查词、打开链接、Finder 路径、排除列表、双语界面。
